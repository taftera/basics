## Create projects
- **git init**
  - create "README.md"
- **git remote add** *REMOTE-NAME* *URL*
- **git push --set-upstream** *REMOTE-NAME* **master**

## Upload updates

- **git status**
  - check if something is "modified".
- **git add** *FILENAME.ext*
- **git commit** -m "*comment about the upload*"
- **git push**
  - to upload content with credentials.

## Untracked
- **git add** *<file>*
  - to include it
